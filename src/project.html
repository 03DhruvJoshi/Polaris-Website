Anipedia

AniPedia is a didactic Android-based Application dedicated to preschool children, and its purpose lies in aiding the following children to learn the appearance, names and classifications of various types of animals. Polaris Solution was in charge in full-stack development of the application, where Android Studio was used, comprising Java & Kotlin languages to develop the architecture of the application, and HTML and XML for the graphic design of the application.
